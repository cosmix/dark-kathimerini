# Dark Kathimerini

A refined dark theme for [kathimerini.gr](https://www.kathimerini.gr/), built as a [Stylus](https://add0n.com/stylus.html) UserCSS style. Proper contrast on Greek text, inverted monochrome K logo, calmed image placeholders, accent blue preserved. Covers the homepage, articles, live blogs, the eortologio, weather, newsletters, subscriptions and static pages.

## Install

1. Install the [Stylus](https://add0n.com/stylus.html) browser extension.
2. Open the raw stylesheet — Stylus shows its install screen:

   https://raw.githubusercontent.com/cosmix/dark-kathimerini/main/dark-kathimerini.css

The style ships an `@updateURL`, so Stylus' **Check for updates** pulls new versions (bump `@version` on each change).
